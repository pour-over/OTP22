## What is Shadytel?

**Shadytel** is a "fake telco" collective from the hacker and phone phreaking scene. It's associated with:
- **ShmooCon** — Annual hacker conference in Washington, DC (January)
- **DEF CON** — Annual hacker conference in Las Vegas (August)
- The broader Pacific Northwest hacker community, especially Seattle
- The **Connections Museum** in Seattle — a working collection of vintage telephone switching equipment (Strowger, crossbar, ESS switches, etc.)

Shadytel members have deep operational knowledge of real telephone systems: routing, switching, provisioning, billing, signaling. They build and maintain actual telephone infrastructure as a hobby/art project.

## The Evidence

### Level 3 Infrastructure Depth

Nearly every original run phone number was provisioned on **Level 3 Communications** (now Lumen Technologies) wholesale SIP trunking. All end in `0004`. This is not consumer-grade VoIP — this is professional telecom carrier infrastructure.

Provisioning Level 3 DIDs at this scale requires:
- A Level 3 wholesale account (requires commercial relationship)
- Knowledge of how to request specific last-four-digit patterns across multiple area codes
- Understanding of how Level 3 routes traffic

This is exactly the kind of knowledge that Shadytel members would have.

### CLLI Code Reference

Agent code `303` on the original system plays: *"You have reached the Time-Warner switch in Aurora, Colorado. The switch CLLI is A U R S C O F W R S 0"*

CLLI codes (Common Language Location Identifiers) are the internal identification system for telephone central office equipment. Knowing that Time-Warner's Aurora CO switch has CLLI `AURSCOFWRS0` is not general knowledge — this is phone phreak trivia.

### Bell System Conventions

Throughout the original run, the game uses authentic Bell System terminology:
- "Class six" numbers (requiring the Cleveland Direct calling card)
- Conference bridge architecture
- The Grover Cleveland March as code `*` (a real telephone culture reference)
- Code `2600` (the classic phone phreak frequency, also the name of the seminal hacker magazine)
- ITA2/TTY/TDD integration
- Message Desk operating as an authentic operator service simulation

### Calling Card Systems

The Cleveland Direct calling card system is modeled exactly on real prepaid calling card infrastructure. The card design, the PIN entry flow, the "class six" number concept — these reflect genuine knowledge of how prepaid calling card systems were architected.

### The Name "West / Frederica"

The original number `303-309-0004` was nicknamed "West" and "Frederica." In telephone routing, "West" is a meaningful geographic designation. "Frederica" is less obvious — but Level 3 has routing nodes and trunks with internal names that would be known to people with access to routing tables.

### Conference Bridges at `555` and `559`

Real operations conferences are held on conference bridges. The choice of `555` (ADHOC) and `559` (OPS CONF) reflects authentic understanding of how operations centers use telephony.

If OTP22's creators are Shadytel-connected, the Connections Museum is where the physical telephone culture that informs the game is maintained and celebrated.
