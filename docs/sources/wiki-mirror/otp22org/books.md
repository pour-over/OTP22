# Books — otp22.org Wiki Mirror

**Source:** https://otp22.org/doku.php?id=books

Books are real-life objects used to distribute messages, one-time pads, and other materials to agents. In the revival era, physical books serve as **OTP key sources** for decoding XORFS messages — each book contains pages of key material, with agents referencing specific page/line combinations.

Books can be obtained by sending postal mail to the CMCH, NJ PO box (Literature Request). Some are also found in dead drops.

### Slate Deposits in the Intermountain West

**The primary OTP key book of the revival era.** Found in multiple dead drops across the USA:

| Drop | Edition | Distinguishing features |
|------|---------|------------------------|
| Las Vegas drop | 2024 edition | Standard, used as OTP source |
| Ruston drop | 2024 edition | Contains Message Desk code `6824418` |
| Nashville drop | (unknown edition) | Found in Little Free Library |
| Edgewood drop | **2025 edition** | Contains invisible ink writings + two stickers |
| Sunnyvale drop | (unknown edition) | Found in Little Free Library near San Jose |

The book's title directly references the **Slate messages** system (agent codes 301/302/303 on AS32).

**Ruston copy note:** The Ruston copy of this book contained the agent code `6824418` — the code for the Message Desk.

## Original-Run Books (ExcitingBooks)

The original run (2011–2016) used a completely different book system — 275+ absurdist books published by the "ExcitingBooks" Twitter/Tumblr account run by Andrew Filer (RIP). These books appeared in physical dead drops and were used as OTP/XOR key material.

See the original-run [book covers archive](../sashahart/book-covers.md) for the complete catalog.

## How to Obtain Revival Books

Send postal mail to:

```
615 US-9 S #172
CMCH, NJ 08210
USA
```

Addressed to: **Literature Request**

You may request specific book titles. Some books require credentials obtainable through the Drop Agent Catalog mailing process — see [postal mail](special-content.md#postal-mail).
