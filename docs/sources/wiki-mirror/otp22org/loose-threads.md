## Unsolved

### insurance.bin

A large binary file found in the **El Paso drop**. XORFS message **18438** hints that it contains dirt on **Debra Fisher**, the Chief Finance Manager at Emtanon. Contents have not been decoded.

**File location:** https://otp22.org/files/elpaso/

### "Shadow Code Drops"

Found in a letter inside a player's mailed package — an unsolved blob of **512 random bytes**. Origin and significance unknown.

### Agent Code 55559

Found among data leaked by Maria Peters. When dialed:
- Says "Melter"
- Prompts for additional input

The community speculates this may provide access to the Melter fleet telemetry or related systems.

## Historical Loose Threads (Original Run)

The original run left many unresolved threads when it went dormant in 2016:

- **AI Bouerges disappearance** — primary agent for mailbox 16041 vanished in spring 2013; J asked "WHERE IS SHE?" repeatedly. Never resolved.
- **SHIP coordinates** — Arctic fleet trajectory toward Svalbard; ultimate destination never confirmed
- **QX query system** — partially decoded; VECTORCOM hardware manufacturer identified but connection not fully explained
- **ABISM terminal** — extensive decoded content but inner workings not fully mapped
- **Robbybox final messages** — what happened to the original Robert/Robby insider?
