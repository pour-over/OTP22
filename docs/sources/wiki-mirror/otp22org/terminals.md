# Terminals — otp22.org Wiki Mirror

**Source:** https://otp22.org/doku.php?id=terminals

Terminals are 1980s-era text-only computers used throughout Emtanon. They appear in two aspects:
1. **Monitoring** in-universe terminals to gain insight into the organization
2. **Interacting** with the universe via remotely accessible terminals on the Emtanon portal

---

## In-Universe Terminals

### Green Terminal

**Location:** Gibraltar (The Rock), Room 101
**Iris talk group:** Accessible via `16009`
**AS32 code:** `3` (SSTV screenshots)

The primary publicly observable terminal at Emtanon HQ in Gibraltar. The Green Terminal hosts Iris conversations about operational matters, including drop placement discussions.

**Notable events:**
- Pawnee Rock drop placement discussed here ("meant for a rookie agent")

---

### Amber Terminal

**AS32 code:** `10` (SSTV screenshots)

Used for operational communications. Most prominent terminal for drop-related activity.

**Notable events:**
- Live coverage of Ruston drop placement (January 1–5, 2024)
- Revealed Tuktoyaktuk drop location as Arctic
- Wailua drop reveal (November 15, 2025)
- Iris conversation deleted by Desiree Martin to hide Riverside drop details

### OVAL Terminal

**Image:** [oval.png](https://otp22.org/lib/exe/detail.php?id=terminals&media=terminals:oval.png)

The OVAL terminal is unique in that it has a **rudimentary GUI** and responds to mouse inputs. It provides access to an Emtanon system called OVAL.

**Access credentials:** Found in an NFC tag within the [Wailua drop](dead-drops.md#wailua-drop) (November 2025).

**Current contents (as of March 2026):**
- Mainly clones of popular games
- A terminal app that can dial telemetry using the **Hayes AT command set**

## Programs

### `as27`

Available on the OVAL terminal. Accesses AS27 broadcast messages directly from the terminal interface.

### `iris`

Available on all terminals. Provides access to the Iris instant messaging network. Used to monitor conversations between Emtanon personnel and Marine Org vessels.

---

## Remote Access

Terminals at Gibraltar (Green, Amber) are observable via SSTV screenshots broadcast on AS32 codes `3`, `10`, and `11`. The SSTV images show current terminal state and recent Iris conversations.

SSTV decoder: https://otp22.org/tools/ships.html (ship tracker) and standard SSTV software.

The OVAL terminal required specific credentials from the Wailua drop to access remotely.

---

## Original-Run Terminals

The original run used the **ABISM protocol** — a custom modem/terminal system. ABISM terminals had commands including GO, HELP, INDEX, PRINT, READ, WRITE and were accessed via DTMF entry on agent codes. See the [ABISM archive](../sashahart/messages/abism.md) for full details.
