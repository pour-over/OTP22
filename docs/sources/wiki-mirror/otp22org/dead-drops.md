# Dead Drops — otp22.org Wiki Mirror

**Source:** https://otp22.org/doku.php?id=dead_drops

WORM distributes materials to their agents using dead drops — hidden containers in public locations, comparable to geocaching. Players act as counter-operatives by retrieving them first.

> *A dead drop is a method of espionage tradecraft used to pass items or information between two individuals using a secret location. — Wikipedia*

## Individual Drop Details

### Cape May Drop

**Source:** https://otp22.org/doku.php?id=dead_drops:cape_may_drop
**Retrieved:** July 21, 2023 by TonyInPA
**Background:** Drop placed by **Felicity Love** while on the run on the US East Coast, being pursued by Marine Org. She used an improvised dead drop spike made with a **fake sprinkler key box**.

**How found:** The drop site was visible on Felicity's Instagram posts.

**Contents:** Physical materials from Felicity; full contents documented with pictures at the wiki.

### El Paso Drop

**Source:** https://otp22.org/doku.php?id=dead_drops:el_paso_drop
**Retrieved:** August 1, 2023 by GeneralJake
**Placed:** Evening of July 23, 2023

The El Paso drop is one of the most significant revival drops, containing digital files.

**Contents:**

| File | Notes |
|------|-------|
| David_1.jpg | Photo with steganographic data |
| Jose_1.jpg | Photo with steganographic data |
| Maria_1.jpg | Photo — steganography confirmed |
| Maria_2.jpg | Photo |
| Maria_3.jpg | Photo |
| Maria_4.jpg | Photo |
| Matt_1.jpg | Photo |
| Tim_1.jpg | Photo |
| trash_files/IF_FOUND.TXT | Recovery instructions |
| **otp-blue-1.bin** | **1 GB OTP key file** — used for all XORFS decryption |
| insurance.bin | Large binary — believed to contain dirt on Debra Fisher (unsolved) |

**Key significance:** The `otp-blue-1.bin` file is the encryption key for all XORFS messages. The El Paso drop was the foundational drop of the revival crypto system.

### Calgary Drop

**Source:** https://otp22.org/doku.php?id=dead_drops:calgary_drop
**Placed:** September 11, 2023
**Background:** Placed by Marine Org **B team** en route to their ship stationed off the coast of LA. They were instructed to leave the drop in **California** ("CA") but misunderstood the abbreviation as **Canada** ("CA" → "Canada"). Ended up in Calgary, Alberta.

*One of the funnier operational failures in OTP22 history.*
