## About otp22.org

otp22.org is the community wiki for the revival era of OTP22 (2023+). It documents an ongoing alternate reality game centered on WORM/Emtanon — a criminal organization melting polar ice caps, stealing seed vaults, and running mercenary operations. Players act as counter-operatives intercepting dead drops, decoding messages, and disrupting WORM's plans.

The wiki uses DokuWiki at `https://otp22.org/doku.php?id=[page_id]`.

## Key Differences from Original Run

| Feature | Original Run (2011–2016) | Revival (2023+) |
|---------|--------------------------|-----------------|
| Phone backbone | Level 3 (303-309-0004) | Multiple numbers |
| Main message system | 18xxx OTP/XOR | XORFS on AS32 |
| Short broadcasts | Code 22 warnings | Slate messages (301/302/303) |
| Drop delivery signal | SHIP coordinates | SSTV on AS32/Felicity voicemail |
| Terminals | ABISM protocol | Live SSH-accessible terminals |
| Books | ExcitingBooks (275+, Andrew Filer) | 7 revival titles, physically mailed |
| Characters | J, AI Bouerges, Robby, etc. | J + Tala, Felicity, Jeff, Maria, Robby |
