## About This Mirror

303.sashahart.net was the primary community-maintained database for the OTP22 ARG during its original run (2011–2015). It tracked:

- Every phone number and agent code with call counts and dates
- The complete story timeline with audio links
- Physical drop events and item inventories
- Decoded messages (OTP/XOR plaintext, SSTV images, ABISM decodes)
- The full book cover gallery (200+ titles by Andrew Filer)
- All data files from physical drops

The site remains live as of 2026 at http://303.sashahart.net/ and this mirror was scraped for preservation in case it goes offline.

## Audio Archives

All audio recordings are hosted at:
- **http://dialer.otp22.com/** — primary archive (organized by year/month)
- **http://vocaroo.com/** — older recordings (some may be dead links)

The dialer archive uses the naming convention:
`YYYY-MM-DD HH-MM UTC - [phone number] - [agent code].wav`
