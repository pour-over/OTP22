## About AS27

AS27 broadcasts daily organizational directives encoded as ASCII numbers spoken by text-to-speech. The message is a daily bulletin format used for fleet/agent coordination. In the revival era, AS27 is at +1-207-813-2200.

**Discovery (original run):** The number 7208970004 was guessed by appending 0004 to the partial number `720 897` overheard in an AS conference call recording.

## Key Terminology in AS27 Messages

| Term | Meaning |
|------|---------|
| Collector 1 | Primary collection point / vessel |
| Drop Agent [N] | Field agent managing a specific drop route |
| Drop Route [NNNNN] | Numbered physical delivery route |
| MELTR Fleet | Melter ship fleet |
| West Fleet / East Fleet | Geographic subdivisions of the fleet |
| CARGOCOM | Cargo communications / operation phase |
| HF RX | High-frequency radio receive (at specific UTC time) |
| Book | Physical operational handbook (Green Book / Worm Book) |
| Agent 9910 | Specific agent with CARGOCOM responsibilities |
| Agent 2, Agent 14 | Field agents; Agent 2 goes AWOL in late May 2014 |

## Operational Pattern

The AS27 broadcasts reveal a highly structured operational hierarchy:
1. **Daily status requests** to fleet commanders
2. **HF radio schedule announcements** (date/time/frequency for backup comms)
3. **Book-based authentication** (read page X, line Y — confirms agent has physical book)
4. **Agent escalation** (contact HQ → ASAP → all agents read about Agent 2's status)

Agent 2's non-response in late May 2014 is a notable plot point — the broadcasts escalate from "provide results" to "CONTACT HQ ASAP" to "ALL AGENTS READ RE: AGENT 2," suggesting Agent 2 went offline or defected.
