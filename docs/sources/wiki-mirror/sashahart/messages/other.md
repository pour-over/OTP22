## ITA2 / Baudot Messages

ITA2 (International Telegraph Alphabet No. 2 / Baudot) encoded messages were found on multiple numbers.

**2012-10-18–19:** Polish number 48223071061 transmitted ITA2 to 3033090004:
```
DEADMANSWITCHTRIPENTRBELLAGENTCODETOREACTIVATEAGENTSYSTEM
```
*"Dead man's switch: trip entry bell agent code to reactivate agent system"*

This appears to be an automatic trigger mechanism — the system required periodic "alive" signals to prevent automatic lockout.

## The "Forever Beautiful" Motif

Multiple codes reference "Forever Beautiful" — a phrase appearing throughout OTP22:

- Code 0 (original): distorted version of "Millionaires — Talk Shit" with "I love you. We love you. Forever Beautiful."
- Code 3: "You will never understand us. How beautiful we are. Yeyuh. Why you cut our Forever Beautiful?"
- Code 5: "How beautiful we are."
- Code 230: "Understand. The situation. The environment. And the cause. For or beauty forever forever beautiful."
- Code 231: "Beautiful."

Physical: "Forever Beautiful" elephant plushie (Pawtucket, 2012) and "Forever Beautiful" dog plushie (Milbank, 2014)

The phrase appears to be a slogan or motto of the MLPL organization — possibly relating to their stated goal of environmental transformation (melting ice = "forever beautiful" Arctic?).
