## About QX

QX (agent code range) is a query-response system accessed via DTMF. Community members ("players") discovered they could submit queries by calling number 2342220004. The system logged queries and responded — sometimes in-character (as the organization), sometimes with genuine operational data.

**Discovery:** "Accidentally done by users bobdamnit and morningcoffee" (for first QX messages)

## QX System Analysis

The QX system reveals:

1. **It's a real customer service interface** — queries get real operational responses
2. **The organization treats players as actual agents** — responds as if they're field operatives
3. **EMTANON is named as a contractor** (not just a vague reference) responsible for AS36 systems
4. **VECTORCOM** is a physical hardware device with button inputs (R key / 0 key)
5. **Cargocom** = an operation involving physical supplies at the "slug drop" (Region 10)
6. **Red Book** = audio CD format used for physical drop contents
