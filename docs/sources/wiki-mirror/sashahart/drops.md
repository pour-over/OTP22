## Source Code Key

Drop announcements came from specific agent codes on the phone system:

| Code | Meaning |
|------|---------|
| 99280–99299 | Drop coordinate announcements via Code 0 broadcast |
| 16041–16045 | Agent voicemail ordering drop retrieval |
| 18xxx | OTP-decrypted messages with drop coordinates |
| 6871–6879 | OTP Black status codes |
| 4822010112 | Green Book instructions (Wallace, ID drop) |
| Blyind / Nadando / Braco / Smugtrio / L0j1k / excitingbooks | Community members who received shipments directly |
