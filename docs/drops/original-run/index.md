# Original Run Dead Drops (2012–2016)

*~50 drops placed worldwide. 44 confirmed recovered. 6 not found.*
*Source: 303.sashahart.net, player records.*

---

## Overview

Physical dead drops were a core mechanic of the original run. Coordinates were delivered via the agent system phone numbers — usually announced on agent code `16041` or `99xxx` codes, often cross-referenced with 18xxx OTP messages. Players would travel to the coordinates and retrieve containers or hidden items.

**Total drops documented:** ~47 in this index (additional may exist)
**Recovered:** ~44
**Not found:** 3–6

## Notable Drops — Details

### Lake Oswego, OR (2012-09-14)
First physical drop ever found. Contents: CD, "First Ladies Book" — left in Lake Oswego Public Library.

### Montevideo, MN (2012-10-04)
Index card reading `7097000122` — the Iristel number that would become the source for dozens of component drop locations. Pivotal discovery.

### Tapiola, Finland (2012-10-15)
International drop — 4 cubic magnets, 5cm spring, tube. These were physical components. Finland was the first confirmed international drop.

### Henderson, NV (2012-12-04)
Contents: **Iridium SIM card** (held by L0j1k) — an actual satellite communications SIM. **Black box device** (unknown). 3M floppy box. The Iridium SIM is one of the most significant physical artifacts — it represents real functional hardware embedded in the ARG.

### Bangkok, Thailand (2013-02-25)
DVD hidden on the ceiling of a phone booth. Contents: `COLLECTING_VINTAGE_CAPACITORS.iso`. One of the game's most dramatic drop scenarios.

### Kauai, HI × 3 (March 2013)
Three closely spaced drops on Kauai over a few days, each with a different OTP CD (OTP7, OTP8, OTP9) plus one *Designing Compelling Worm Sanctuaries* book copy. The OTP CDs were key material for decoding the 18xxx message archive.

### Malaysia, Taiping (2013-03-24)
Contents: desiccant gel, knob, open tube, spindle — physical components, similar to Tapiola and Pawtucket.

### Sebastopol, CA (2013-10-22) — **drawbars**
Held by **drawbars**. Contents:
- Cleveland Direct calling card #2
- Note reading: *"AUTHENTIC NIGHTLIGHT FOR FUTURE USE"*

This is one of the most-referenced drops in the database because drawbars is the repository's primary compiler.

### Rohnert Park, CA (2014-03-25) — **drawbars**
Held by **drawbars**. Contents:
- FUNCTIONAL TEST FOR AS36 printout
- FUNCTIONAL TEST RESULT REPORT for AS36

AS36 was the remote field installation central to the original run's operational arc. Having functional test documentation for it is significant.

### Missoula, MT (2014-06-26)
Contents: ruler with hex data: *"GROUP 1 9e 0b A8 6b be 14 d8 95..."*

### Milbank, SD (2014-08-08)
Contents: "Forever Beautiful" dog plushie (callback to the 2012 Pawtucket "Forever Beautiful" elephant plushie), noisemaker, Serbian cookie wrapper, Ziploc bag, Tupperware. The "Forever Beautiful" motif (agent codes `3` and `5`) appears across multiple years.
