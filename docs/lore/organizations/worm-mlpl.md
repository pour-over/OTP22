# WORM / MLPL

*"World Organization for the Reform of Metorology [sic]"*

---

## Overview

**WORM** (World Organization for the Reform of Metorology — the misspelling is intentional in-game) is the third of J's organizations. It conducts **Project MLPL** — a multi-pronged covert operation aimed at environmental and infrastructural disruption on a global scale.

Players of OTP22 are positioned as **counter-operatives** working to disrupt WORM's activities from within and without.

---

## Project MLPL

MLPL's known objectives (from the "MLPL OBJECTIVES" document retrieved in the Braco/Sydney drop, March 2013):

### Arctic Ice Melting
WORM operates a **Melter Fleet** (original run: MELTR1, MELTR2, MELTR4) to melt Arctic ice. The original run's entire operational arc — the Svalbard Vault, component assembly, the TRANSP voyages near the Bering Sea — was in service of Arctic operations. The revival-era Bouvet Island construction may be a related project.

Reference: The field installation AS36 ("Alpha Sigma 36") was a remote battery/solar/radio unit involved in supporting Melter fleet operations.

### Seed Vault Infiltration
WORM operations include stealing seeds from seed vaults. **Svalbard Global Seed Vault** (Longyearbyen, Norway) is the presumed target referenced in early 2012 messages: *"PROCEED TO SVALBARD VAULT FOR FINAL ASSEMBLY."*

### Agricultural Sabotage
Corn blight deployment to destroy crops. Referenced in internal MLPL documents.

### Power Grid Sabotage
Triggering power blackouts by exploiting backdoored Emtanon IoT appliances. The backdoor infrastructure is built into legitimate Emtanon consumer products.

## Key Artifacts

**"MLPL OBJECTIVES"** — Document retrieved from the Braco/Sydney drop (March 27, 2013). Held by L0j1k. Describes the organization's stated goals. One of the most significant physical artifacts of the original run.

**"Designing Compelling Worm Sanctuaries"** — Written by "North American Helminthological Society." Physical book found in multiple drops (Kauai HI, Wallace ID, Sofia Bulgaria). Contains OTP material, phone numbers, and operational context. The book is also the source of phone numbers — the Message Desk number `2022042303` was printed inside.

**"Global Response Codes"** — Written by Dr. J. Marvin Blackweather. First named appearance of J. Retrieved in Nadando (Tempe AZ) shipment, February 2013. Held by L0j1k.

## The Agent Network

WORM/MLPL operated a global field agent network. The agent code system (`16xxx` range) tracks individual agents. Known agent codes from the original run:

- `16041` — AI Bouerges (primary agent)
- `16042–16045` — Other agents
- `16001` — Agent directory / AI Bouerges profile
- `16241/16242/16243` — Upload/handshake codes for respective agents

The 2023 revival expanded this: the agent management system at AS32 allows new agents to be registered, and there are now thousands of active agent numbers.

## WORM vs. Players

The game positions players as counter-operatives who have somehow gained access to the agent systems. The tension of the original run is that players are listening in on real operational traffic, retrieving physical components that were meant to reach the Svalbard vault, and communicating with a rogue insider (Robby) who is leaking from the inside.

In the revival, players can interact with Emtanon terminals, intercept Iris messages, use the Path Controller to manipulate physical systems (semaphores, ship controls), and actively intervene in WORM operations.
