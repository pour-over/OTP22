## Original Run

### The First Drop (September 14, 2012)
The first physical dead drop was found in Lake Oswego, Oregon — a CD and a "First Ladies Book" left in the public library. This established the game's physical component: real objects at real coordinates, discovered by following phone system instructions.

### Multi-Drop Broadcast (October 15, 2012)
The most significant early event. Code 0 broadcast simultaneous drop coordinates for Santa Cruz CA, Tapiola Finland, Honolulu HI, Chicago IL, Pawtucket RI, and Ibaraki Japan — along with a message directing agents to *"PROCEED TO SVALBARD VAULT FOR FINAL ASSEMBLY."* This was the first indication of the game's international scope and Arctic narrative.

### Agent 16041 Goes Active (November 21, 2012)
Agent code `16041` came online, establishing the agent voicemail and instruction system that would be the primary player interface for years. Agent AI Bouerges became the central character of the original run.

### Henderson Drop — Iridium SIM (December 4, 2012)
The Henderson NV drop contained an Iridium SIM card — hardware for actual satellite communications. Real functional technology embedded in the game's physical layer.

### Designing Compelling Worm Sanctuaries (March 2013)
Multiple drops in March 2013 (Kauai HI, Wallace ID) distributed copies of this book — the core narrative artifact of the original run. Written "by the North American Helminthological Society." Contains OTP material and a phone number (Message Desk) inside.

### The Bangkok DVD (February 25, 2013)
A DVD hidden on the ceiling of a phone booth in Bangkok, Thailand. Contents: `COLLECTING_VINTAGE_CAPACITORS.iso` — referencing an ExcitingBooks title. One of the most dramatic drop locations of the original run.

### Robby Goes Rogue (September 22, 2013)
Robert "Robby" Bramble first messaged the community via the Robbybox voicemail (`50565732`). Over the following weeks he described the internal systems in detail, offering to share "something of great interest." J's response when Robby escaped: *"I CANT BELIEVE YOU LET THAT STUPID LITTLE BASTARD GET AWAY."*

### IRISLATCH / Melter Fleet Standby (August 2013)
"Operation IRISLATCH" messages announced the Melter fleet (MELTR1, MELTR2, MELTR4) in position. This represented the original run's peak operational intensity.

### Bulgarian Drops (September 2013)
Drops in Dimitrovgrad and Sofia, Bulgaria — the farthest-flung European drops of the original run. The Dimitrovgrad CD contained a video with DTMF tones that decoded to additional phone numbers.

### LONGVAULT (December 2013)
A named operation executed in late 2013 involving agent D heading to a shore location. Possibly the original run's climax operational sequence.

### Bering Sea Encounter (June–July 2014)
Messages documented the TRANSP/Melter fleet in the Bering Sea encountering Russian Navy vessels. A "compromising event" is reported: *"If operating in Region 9, contact HQ ASAP."*

### End of Original Run (2016)
The game went dark. Agent systems stopped responding. No formal conclusion was announced.

## Revival Era (2023–present)

### Revival Launch (May 2023)
OTP22 relaunched with significantly expanded infrastructure: new agent system AS32, emtanon.com web portal with real terminal emulators, marineorg.com, and a new narrative cast.

### Gough Ceremony (September 24, 2023)
A major in-universe event broadcast over all agent systems from Gough Island. Disrupted communications across the system. The first major synchronized event of the revival.

### Message Desk Rediscovery (January 7, 2024)
Agent code `6824418` — the Message Desk originally operated by Andrew Filer — was rediscovered via a code in the Ruston drop book. The Desk responded. Andrew Filer's creation was now part of the revival.

### Operation Spice (January 2024)
WORM operation in Lima, Peru targeting counter-operatives. Handler: Maria Peters. Complete failure:
- MO3 lost
- Crew captured by Peruvian cartel
- MO5 crew defected to counter-operatives

### Bouvet Island Acquisition (January 2024)
North Korean missiles acquired at Bouvet Island. Construction of unknown facility begins March 2024. Bouvet is one of the world's most remote and inhospitable locations — a subantarctic volcanic island.

### FACILITY Redemption (February 2024)
Players provided Path Controller access (via codes from Felicity Love's voicemail and mailed materials) to flip semaphore bytes controlling FACILITY's cell locks:

```
0xFF → 0x00 (all cells unlocked)
```

Detainees obtained chemical weapons, killed the bridge crew, and took control of the prison ship. Morgan "The Edge" Fellman was freed. FACILITY sailed for the Pacific Ocean.

### Aqua Terminal Goes Dark (February 21, 2024)
AIR1 and MO5 took the Aqua Terminal (Hilo, Hawaii) offline to hide evidence from the Operation Spice fallout. Restored June 7, 2024.
