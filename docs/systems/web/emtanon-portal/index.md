# Emtanon Portal

*Web-based terminal emulator access at emtanon.com.*

---

## Overview

The Emtanon portal at **https://emtanon.com** provides browser-based access to the Emtanon terminal network. Terminals simulate 1980s-era text-only computers.

Access requires credentials, obtainable by mailing Access Request form `AC-P-REQ-01` to the CMCH NJ PO Box.
