# Iris Messaging System

*Emtanon's real-time internal messaging platform.*

---

## Overview

**Iris** is the internal instant messaging system used throughout the Emtanon organization. It's accessible from every terminal and from mobile units (ships, aircraft). Talk groups are numeric; negative group numbers indicate mobile/field units.

Players can access Iris via the terminal `iris` program (run from any Emtanon terminal). Intercepted Iris conversations have been a key source of narrative intelligence in the revival.

---

## Access

From any Emtanon terminal, run:
```
run iris
```

Or via the `iris` program entry in the terminal program list.

---

## Intercepted Conversations

The community has documented numerous Iris intercepts that reveal organizational drama, operational orders, and character development. Key intercepts are archived on the otp22.org wiki under the Iris section.

Notable events visible through Iris:
- Operation Spice planning and aftermath
- FACILITY Redemption coordination between AIR1 and MO5
- J's day-to-day communications
- Esprit Boursicot's terminal testing logs

---

## Original Run Equivalent

The original run did not have a real-time messaging system accessible to players. The `18xxx` OTP messages were the narrative equivalent — but delivered via the phone system as asynchronous audio. Iris represents a significant expansion of player access to in-universe communications in the revival.
