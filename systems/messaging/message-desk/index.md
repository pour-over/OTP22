# Message Desk

*Agent code `6824418` on AS32. Operated by a real human.*

---

## Overview

The Message Desk is one of OTP22's most distinctive features: a phone code staffed by an actual person who responds to numeric inputs with location data, coordinates, and cryptic messages.

In the original run it was operated by **Andrew Filer** ("Troy") on the Windstream number `202-204-2303` (Okinawa). In the revival, it moved to agent code `6824418` on AS32 (`+1-606-722-0004`).

---

## Original Run (2012–2014)

**Number:** `202-204-2303` (Okinawa, Windstream)
**Operator:** Andrew Filer ("Troy")

The Message Desk number was printed inside the front cover of *Designing Compelling Worm Sanctuaries*. It was also referenced in IFFOUND.TXT files on CDs from multiple drops:
- Smugtrio (2013-04-12)
- NYC (2013-05-09)
- Pennsylvania/Carlisle (2013-05-23)
- Dimitrovgrad, Bulgaria (2013-09-06)

Known original run responses:
- `303` → coordinates
- `445` → "Diego Garcia"
- `660` → "Out of service"
- `68721` → `022-6993` (a phone number)

When Troy (Andrew) answered, he responded in character as a WORM/Emtanon communication system. The experience of reaching him — hearing a real human voice on what otherwise was an automated system — was considered one of the most memorable moments of the original run.

The Desk went quiet. Andrew Filer passed away.

---

## Revival Rediscovery (January 7, 2024)

The Message Desk reappeared. A code in the **Ruston, WA drop book** (2024) pointed to agent code `6824418` on AS32. Players dialed it and got a response.

The Desk now operates on AS32 instead of a separate number, but the experience is consistent with the original: numeric inputs, human-feeling responses, location data and coordinates.

---

## How to Use

1. Dial AS32: `+1-606-722-0004`
2. Enter agent code `6824418`
3. Enter a numeric query via DTMF
4. Listen for response

---

## Context

The Message Desk exists at the boundary of the game's fiction. When you call it, you may be speaking to a person who is genuinely there — a member of the production team maintaining Andrew's legacy. Or it may now be automated with carefully designed responses that preserve the feel.

Either way, it is the game's most human interface: a phone number that listens, and answers.
